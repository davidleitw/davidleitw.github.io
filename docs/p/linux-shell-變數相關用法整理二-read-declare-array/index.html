read

讀取從鍵盤輸入的變數，常常會出現在 shell script 之中

$ read [-pt] 變數名稱
// 參數
// -p 後面接提示字元
// -t 後面接等待時間，超過時間沒輸入則取消指令
$

declare

可以宣告變數的類型，我們上面提到有關變數的操作很多都只是單純的字串，但有些場合我們需要整數或者陣列時就要使用這個指令去指定變數的類型

1
2
3
4
5
6
7
8

$ declare [-aixr] 變數名稱
// 參數
// -a 宣告陣列型態的變數
// -i 宣告整數型態的變數
// -x 同 export 指令，宣告的變數變成全域變數(可以讓子程序讀取)
// -r 宣告 readonly, 不可更改變數內容，也無法被 unset 取消
// -p 列出變數資料型態
$
